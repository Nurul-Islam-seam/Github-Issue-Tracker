1️⃣ What is the difference between var, let, and const?
Ans: 
"var" is function-scoped and can be redeclared, "let" is block-scoped and can be reassigned, and "const" is block-scoped but cannot be reassigned.


2️⃣ What is the spread operator (...)?
Ans:
The spread operator (...) is used to expand or copy elements of an array or object.

3️⃣ What is the difference between map(), filter(), and forEach()?
Ans:
map() creates a new transformed array, filter() returns elements that match a condition, and forEach() just loops through the array without returning a new one.

4️⃣ What is an arrow function?
Ans:
An arrow function is a shorter way to write functions in JavaScript using =>.

5️⃣ What are template literals?
Ans:
Template literals are strings written with backticks ( ) that allow variables inside using ${}.
